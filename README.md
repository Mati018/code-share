# Code Share: Realtime Collaborative Code Editor

### Prerequisites

#### For running locally

- Node.js (v22.11.0)
- npm (7.24.2)

## Tech Stack

- React.js
- Node.js
- Express.js
- Socket.io
- CodeMirror
- React-Toastify

## Installation

### Running Locally

1. Clone this repository and cd into it
2. Run `npm install` to install the dependencies
3. Create .env file in the root folder and and add credentials for REACT_APP_BACKEND_URL, SERVER_PORT.
4. To start the react app client run `npm start` in one terminal
5. To start the server run `npm run server:dev` in another terminal
6. Go to `http://localhost:3000` to view the app
